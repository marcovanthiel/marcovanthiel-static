# Werkgever-logo's

Drop hier de logo's van de werkgevers/opdrachtgevers als PNG-bestanden.
Zodra een bestand bestaat verschijnt het automatisch op de **Ervaring**-tab
in plaats van het letter-monogram.

## Tips

- **Formaat**: vierkant of bijna-vierkant werkt het beste (kaart is 96×96).
- **Type**: PNG met transparante achtergrond geeft het mooiste effect (het
  logo lijkt dan direct op de witte kaart te zweven).
- **Resolutie**: minimaal 192×192 (2× retina). Hoger mag.
- **SVG**: kan ook — verander dan de bestandsnaam-extensie in `index.html`
  van `.png` naar `.svg`.

## Fallback

Zolang een bestand niet bestaat (of niet laadt) toont de pagina automatisch
een letter-monogram op een gekleurde top-strip (gekleurd per sector: zorg,
justitie, publiek, bedrijfsleven). Geen extra code nodig.
